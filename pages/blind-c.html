Blind C

Introduction


Blind C is a unique programming competition where you have to write the program in c language for the given problem with the monitor kept off. Problem statement will be given on the spot to the participants.

Purpose


This activity mainly tests your concentration, accuracy and programming skills.

General Guideline


  1. Mobile phone strictly prohibited

  2. Use of internet is prohibited

Rules & Regulations


  • Each participant has to write the code for the given problem, with MONITOR OFF.

  • Participants whose code gets executed will be given higher preference.

  • The second preference will be given to code with the least number of errors.

  • In case of a tie in the number of errors, TIME of SUBMISSION will be considered.

  • This is an individual event. The compiler provided will be Turbo-C.

  • This event consists of 2 rounds.
    • Round-1 Contains “C” Programs of medium complexity.
    • Round-2 Contains “C” Programs of high complexity.

  • Shortlisted participants of Round -1 will appear for Round-2 for which above rules are applicable.
